# TinyGPT 🧠

**TinyGPT** is my personal journey into understanding how GPT models *actually* work — by building one from scratch. 🚀

This project is a **hands-on exploration** of transformer architectures, where I'm learning by doing. No black boxes. No magic. Just code and curiosity.

---

## 📚 What I'm learning along the way:
- 🔄 How transformers process and generate text  
- 👁️ The mechanics of **self-attention**  
- ⚙️ What it takes to train a language model — **even a tiny one**  

> This isn't meant to be a production-ready model or a competitor to existing frameworks.  
> It's a **learning tool** that happens to be public. 🛠️

---

## 🧪 Training the Model

### 📦 Prerequisites
Before running the model, you'll need to download a training dataset.  
For example, you can use **Shakespeare's complete works** 📜.

### ⚙️ Dataset Configuration
**⚠️ Important:** By default, the model expects your training file to be named: `t8.shakespeare.txt`


### 🛠️ Custom Datasets
If you wish to use a different dataset or filename, you'll need to **modify the code** accordingly.
